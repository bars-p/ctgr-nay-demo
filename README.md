# CTGR NAy Demo Project

## Building Parts

- VueJS 3
- Pinia
- vue-router
- Vuetify 3
- MapboxGL
- DeckGL

## Project setup

```
# yarn
yarn

# npm
npm install

# pnpm
pnpm install
```

### Compiles and hot-reloads for development

```
# yarn
yarn dev

# npm
npm run dev

# pnpm
pnpm dev
```

### Compiles and minifies for production

```
# yarn
yarn build

# npm
npm run build

# pnpm
pnpm build
```
